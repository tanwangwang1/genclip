# AI 视频生成模型定价说明

本文档说明了 VideoFly 平台 AI 视频生成模型的定价策略与成本分析。

---

## 📊 定价第一步：成本与利润分析

### 核心原则

在确定任何定价之前，必须先回答三个问题：

1. **我的成本是多少？** - 每个 AI 模型的 API 调用成本
2. **我想要多少利润？** - 目标毛利率（建议 50-70%）
3. **用户愿意付多少钱？** - 市场价格与价值感知

### 成本计算公式

```
积分消耗 = ceil(API成本(美元) × 倍数)
毛利率 = (售价 - 成本) / 售价 × 100%
```

### 倍数选择指南

| 策略 | 倍数 | 毛利率 | 适用场景 |
|------|------|--------|----------|
| 保守 | 10x | 90% | 高端市场，强调品质 |
| 平衡 | 5-7x | 80-86% | 竞争激烈市场 |
| 激进 | 2.5-3x | 60-67% | **推荐**，平衡获客与利润 |

### VideoFly 实际定价

基于 Evolink.ai 进货价 **¥0.1/积分**：

- **Veo 3.1**: 10 积分 = ¥1.00 ≈ $0.14 成本
- **售价倍数**: 暂按 1:1 成本定价（可调整）
- **目标毛利率**: 50-70%（建议后期调整）

### 定价策略要点

✅ **订阅为主，一次性为辅** - 订阅保证持续收入
✅ **年付折扣激励** - 通过赠送积分实现 20-35% 折扣
✅ **分层定价** - 满足不同预算用户（Basic/Pro/Ultimate）
✅ **新用户赠送** - 仅赠送低成本模型（如 Sora 2）1 个视频

⚠️ **关键提醒**：
- 成本会变动，需要定期监控实际 API 成本
- 用户使用习惯会影响实际毛利率
- 建议上线后 1-2 个月根据数据调整

## 定价策略

- **换算标准**: 1 产品积分 = 1 Evolink Credit
- **价格处理**: 所有 Credits 向上取整到整数
- **利润率**: 按成本定价（1:1），暂不额外加价
